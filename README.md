# 🏰 阿瓦隆 · 聖杯傳奇 — 線上連線版

**Avalon Online** 是一款基於《阿瓦隆》桌遊規則的全端即時語音多人遊戲，支援 5–10 人遠端連線對戰，無需任何後端伺服器，完全 Serverless 架構。

🌐 **線上遊玩**：[avalon-phi-tan.vercel.app](https://avalon-phi-tan.vercel.app)

---

## ✨ 功能特色

| 功能              | 說明                                          |
| ----------------- | --------------------------------------------- |
| 🎭 **角色發牌**   | 隨機分配身分，隨人數自動升級為進階角色        |
| 🔒 **身分信箋**   | 點擊翻牌顯示你的陣營視野與機密情報            |
| 🎙️ **即時語音**   | WebRTC P2P 語音通話，無需第三方服務           |
| 🗳️ **任務投票**   | 提案→公開表決→任務出征→秘密投票，完整遊戲流程 |
| 🗡️ **刺殺梅林**   | 好人獲勝後，刺客可指認梅林逆轉勝負            |
| 🔊 **發言系統**   | 順/逆時針輪流發言，含計時器與波形顯示         |
| 🤖 **Bot 填充**   | 房長可新增 Bot 玩家進行單人測試               |
| 🛡️ **防作弊鎖定** | 有真實玩家加入時，自動鎖定角色視角切換        |
| 📱 **全裝置適配** | 手機與桌機皆可流暢遊玩                        |

- 🔮 **派西維爾**：看見梅林與莫甘娜的混合名單（無法分辨）
- 🧙‍♀️ **莫甘娜**：假冒梅林出現在派西維爾的視野中

## 🏗️ 技術架構

```
前端：Vue 3 + Tailwind CSS v4 (Vite)
即時通訊：Supabase Realtime Broadcast (遊戲狀態同步 + WebRTC 信令)
語音通話：WebRTC P2P (直接點對點，無需語音伺服器)
部署：Vercel (Serverless)
```

## 📜 遊戲流程

```
建立/加入房間 → 等待玩家加入 → 房長開始遊戲
  → 角色發牌 → 5 回合任務：
      1. 領袖提案 (選出征隊伍)
      2. 全員公開投票 (通過/拒絕)
      3. 出征隊伍語音討論
      4. 出征隊伍秘密投票任務成敗
  → 好人 3 勝：刺客指認梅林
  → 邪惡 3 勝 或 指認梅林成功：邪惡獲勝
```
