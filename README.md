# 🏆 阿瓦隆聖杯傳奇 - Avalon Online (連線模擬版)

這是一個基於 **Vue 3 + Vite + Tailwind CSS v4** 構建的《阿瓦隆 (Avalon)》桌上遊戲線上版前端邏輯模擬測試器。專案內置了完整遊戲流程、語音發言控制台、座次排序以及全套的 Bot 自動化玩家，支援單機開發、視角切換除錯及完整的對局流程演練。

---

## ⚔️ 核心功能特色

1. **圓桌語音討論系統**
   - **出征優先討論**：在領袖指派隊伍後，強制開啟全員限時發言討論階段。
   - **首位決定順序**：第一位發言者（領袖）可自訂發言方向（順時針 ➡️ / 逆時針 ⬅️）。
   - **一分鐘發言限制**：每位玩家限時一分鐘（60秒），支援提前點擊「Pass」交棒給下一位。
   - **語音模擬與動畫**：內置 SVG 語音波形動態動畫，模擬麥克風啟用狀態。Bot 發言時會播放低音 Hum 音效，並自動於 5 秒後 Pass 以便利單機測試。

2. **完整核心遊戲流程**
   - **任務提議階段**：當前期領袖挑選特定數量的玩家出征。
   - **公開投票表決**：全員討論後，公開投下「同意 👍 / 反對 👎」票。若過半同意則隊伍出發，否則否決次數 +1 並移交領袖。
   - **秘密投票執行**：出征成員秘密投票（任務成功 🛡️ / 任務失敗 👿）。好人限制只能投成功；壞人可任意選擇。
   - **歷史出征查看**：在左側「聖杯任務進度板」，點擊 R1 ~ R5 節點即可展開查看各回合的歷史出征成員及成敗結果。
   - **刺客刺殺梅林**：好人方贏得 3 個任務時觸發，刺客必須指認梅林，若指認成功由邪惡逆轉勝。

3. **單機除錯面板**
   - 點擊左側圓桌成員列表的任意玩家，可**切換至該玩家的陣營視角**，藉此代替其在投票、任務秘密投票或刺殺梅林時做出決策，極度便於單人開發測試。

## 🛠️ 開發與建構指令

### 1. 安裝依賴
```bash
npm install
```

### 2. 本地開發伺服器
```bash
npm run dev
```

### 3. 編譯與打包 (Production Build)
```bash
npm run build
```

### 4. 預覽打包後的產物
```bash
npm run preview
```
